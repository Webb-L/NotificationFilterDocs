配置飞书

在配置飞书之前需要创建一个企业账号，只有企业账号才能使用飞书表格API。

注册账号​

打开飞书网站：https://www.feishu.cn/。

点击免费试用按钮后输入手机号再同意协议点击下一步按钮。

填写企业信息后再点击下一步按钮。

填写企业信息后再点击下一步按钮。

设置密码也可以跳过该步骤。

创建应用​

打开飞书开放平台：https://open.feishu.cn/app/。

点击创建应用按钮。

填写应用基本信息。

点击刚才创建的应用。

点击前往编辑按钮。

点击综合信息右侧的图标。

点击管理后台主页上方的图片然后选择图片再点击保存按钮。

点击保存按钮。

设置权限​

选择“查看、评论、编辑和管理电子表格”权限。

点击批量开通按钮。

点击确认并前往创建应用版本按钮。

发布版本​

点击创建版本按钮。

输入版本信息后点击底部的保存按钮。

点击申请线上发布按钮。

同意申请线上发布​

打开管理后台。

打开管理后台应用审核页面在点击右侧的审核按钮。

点击通过按钮。
