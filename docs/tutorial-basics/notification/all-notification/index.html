所有通知

页面介绍​

通知列表​

  • 过滤通知。
  • 显示过更多类型的通知。
  • 过滤成功的规则。

过滤通知​

点击右上角的按钮打开过滤通知页面。

通知统计​

右往左滑动切换到通知统计图表。

点击这里的应用会在过滤通知页面新增过滤应用。

查看通知​

  • 点击通知信息右侧按钮打开通知菜单。

  • 查看通知更多信息（只能查看该应用最新的30条通知）。
